# 上传到GitHub指南

## 准备工作

### 1. 确保已安装Git
```bash
git --version
```

如果未安装，请下载安装：https://git-scm.com/downloads

### 2. 登录GitHub
打开浏览器访问：https://github.com/
登录你的账号

### 3. 创建仓库（如果还没有）
1. 点击右上角 `+` 号
2. 选择 `New repository`
3. 仓库名：`mimasm4`
4. 点击 `Create repository`

## 上传步骤

### 方法1: 使用Git命令行

```bash
# 1. 进入项目目录
cd "d:\py文件保存\密码学\第二个项目SM4加密工具\SM4_手机版"

# 2. 初始化git仓库
git init

# 3. 添加所有文件
git add .

# 4. 提交
git commit -m "Initial commit: SM4手机版加密工具"

# 5. 关联远程仓库
git remote add origin https://github.com/你的用户名/mimasm4.git

# 6. 推送到GitHub
git push -u origin main
```

### 方法2: 使用GitHub Desktop（更简单）

1. 下载安装GitHub Desktop: https://desktop.github.com/
2. 打开GitHub Desktop
3. 登录你的账号
4. 点击 `File` -> `Add Local Repository`
5. 选择文件夹：`SM4_手机版`
6. 点击 `Publish repository`
7. 仓库名填写：`mimasm4`
8. 点击 `Publish`

### 方法3: 使用网页上传

1. 打开你的仓库：https://github.com/你的用户名/mimasm4
2. 点击 `Add file` -> `Upload files`
3. 拖拽所有文件到上传区域
4. 填写提交信息
5. 点击 `Commit changes`

## 遇到问题？

### 如果提示身份验证失败：
1. 使用个人访问令牌代替密码
2. 在GitHub设置中创建Token：Settings -> Developer settings -> Personal access tokens
3. 勾选 repo 权限

### 如果遇到中文文件名问题：
```bash
git config core.quotepath false
```

### 如果遇到换行符问题：
```bash
git config core.autocrlf true
```
